# PRODJET Conditional Waitlist Form

A simple React application for the PRODJET waitlist.

## Features

- Select user type (Student, Brand, or College)
- Dynamic form fields based on selected user type
- Basic form validation
- Success screen after submission
- Responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS

## Run Locally

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
